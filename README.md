# Home-SOC-Environment
This project consist of security research conducted in my home environment between two Linux Vms. This repository will breakdown what tools i used and why, what my architecture looks like, what I've detected and investigated so far. and what I'm building toward.
The goal of this project is to research, build, investigate, and configure a SIEM (Security Information and Event Management) system. To begin, I set up two separate virtual machines running Ubuntu. I installed the Wazuh Manager on the “siemstress” virtual machine, which serves as the central SIEM server. I then deployed and activated the Wazuh agent on the “insecure” virtual machine, which acts as the monitored endpoint. This configuration allows the SIEM to collect and analyze security events generated from the agent system.
